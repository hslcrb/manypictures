# CONTRIBUTING.md - Many Pictures Contribution Guidelines / 기여 가이드라인

'Many Pictures' 프로젝트에 관심을 가져주셔서 감사합니다! 기여를 희망하신다면 아래 가이드를 따라주세요.
Thank you for your interest in the 'Many Pictures' project! If you wish to contribute, please follow the guide below.

## 기여 방법 / How to Contribute

1. **이슈 보고 / Reporting Issues**: 
   - 버그나 새로운 제안은 GitHub Issues를 통해 보고해 주세요.
   - Report bugs or new suggestions via GitHub Issues.
2. **코드 기여 / Code Contributions**:
   - 이 프로젝트는 **Antigravity (Gemini)** 에이전트에 의해 고도로 최적화되어 있습니다.
   - 모든 기여는 기존의 "괴물급" 성능 지표와 한영 병기 규칙을 준수해야 합니다.
   - This project is highly optimized by the **Antigravity (Gemini)** agent.
   - All contributions must comply with the existing "Monster" grade performance metrics and KR/EN bilingual rules.

## 기여 절차 / Contribution Process

1. 저장소를 포크(Fork)합니다. / Fork the repository.
2. 새로운 브랜치를 생성합니다 (`feature/awesome-feature`). / Create a new branch.
3. 한영 병기 주석과 함께 작업합니다. / Work with KR/EN bilingual comments.
4. `make clean && make` 빌드 성공을 확인합니다. / Ensure the build succeeds.
5. 풀 리퀘스트(Pull Request)를 제출합니다. / Submit a Pull Request.
