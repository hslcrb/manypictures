# SECURITY.md - Many Pictures Security Policy / 보안 정책

## 지원되는 버전 / Supported Versions

이 소프트웨어는 최신 버전만 보안 패치를 지원합니다.
Security patches are only supported for the latest version of this software.

| 버전 / Version | 지원 여부 / Supported |
| :------------- | :-------------------- |
| v1.0.x         | ✅ YES                 |
| < v1.0.0       | ❌ NO                  |

## 취약점 보고 / Reporting a Vulnerability

취약점을 발견하셨다면 직접 공개하지 마시고, 아래 보안 연락처로 보고해 주시기 바랍니다.
If you find a security vulnerability, please do not disclose it publicly. Report it to the security contact below.

- **보안 연락처 / Security Contact**: security@rheehose.com
- **Supreme Security Standard**: This software follows the "Monster Grade" security audit protocol, ensuring mathematically validated memory safety in pure C.
- **대응 절차 / Response Process**: 
  - 보고 접수 후 48시간 이내에 확인 메일을 발송합니다.
  - 보안 취약점으로 확인될 경우, 우선적으로 패치를 진행하고 GitHub 권고문을 통해 공개합니다.
  - We will send a confirmation email within 48 hours of receiving the report.
  - If confirmed as a security vulnerability, we will prioritize patching and disclose it via GitHub Advisories.
